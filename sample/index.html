CJLZoom

Lightweight,easy to use JQuery image zoom

[Download (4kb)] | Fork me on Github

Installation

Requires jQuery 1.6+

<link rel="stylesheet" href="cjlZoom.css">
<script src="jquery-1.10.2.min.js"> </script>
<script src="cjlZoom-min.js"> </script>

Basic Usage

Basic Zooming

<script> $(document).ready(function(){ //directly pass the image element $('#zoom1').cjlZoom(); }); </script>

HTML

<img id="zoom1" src="images/watch.jpg">


For Optimal Performance

Google Page Speed Test and GTMetrix will always suggest about having a serve scaled images. Large images being scaled to little would result in page speed penalty.It would be reasonable to have serve scaled images and thus HD pictures will be used as a zoom image.In your image element,just provide an attribute "data-img-large", that points to the HD image

API

Options & Callbacks
  • width [number] 200
    Sets the width of the zoom container
  • height [number] 200
    Sets the height of the zoom container
  • style [String] lens
    Sets zoom style
  • initZoom [number] 2
    Sets initial Zoom when the image is hovered
  • minZoom [number] 1.5
    Sets minimum image size of the zoom image when scrolled
  • maxZoom [number] 10
    Sets maximum image size of the zoom image when scrolled
  • distance [number] 10
    (Only for zoom style outside).Sets the distance between the image element and the zoom container
  • onShow [function]
    triggers before the zoom container is shown
  • onClose [function]
    triggers before the zoom container is closed
